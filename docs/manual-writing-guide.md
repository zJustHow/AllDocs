# 操作手册编写规范

面向**撰写并上传至 AllDocs** 的操作指南、维修手册编写人员。系统架构与入库配置见 [README](../README.md)。

**核心原则：** 可检索的主体是**文字**；插图/表格通过**图号**与正文步骤配对，问答时展示在对应步骤下方，而非堆在文末。

---

## 1. 推荐格式

**正式手册请用带完整书签、可选中文本层的 PDF**（非纯扫描件）。PDF 同时提供章节结构、页码、版面坐标与内嵌图/表，检索与插图对齐效果最好。


| 场景     | 格式                         |
| ------ | -------------------------- |
| 正式手册   | **PDF**（首选）                |
| 内部草稿   | Word → 定稿后导出 PDF 上传        |
| 技术文档   | Markdown → 含大量插图时仍建议输出 PDF |
| 短文、无结构 | TXT / HTML（不适合长篇手册）        |


Word 用内置「标题 1–6」样式；Markdown 用 `#` 标题。单独上传的图片仅整图 OCR，无章节结构。

---

## 2. 内容与排版

- **文字优先**：步骤、参数、报警码、警告等写成正文或文字表，不要用图片替代。
- **图片作补充**：外观示意、接线图、界面截图等看图更直观的内容；一图一注，不重复粘贴相同大图。
- **表格**：用**可编辑的文字表**（Word 表格、PDF 矢量表），不要用截图、扫描图或整页图片充当表格。单元格内不要嵌图片。
- **步骤**：用编号列表书写，关键信息不要只印在图上。
- **页眉页脚**：只放文档名、版本、页码；参数与步骤写在正文主体（上下约 8% 边距会被过滤）。


| 推荐                 | 避免                |
| ------------------ | ----------------- |
| 文字步骤 + 图注 + 正文图号引用 | 纯扫描件、整页长图         |
| 图号写在步骤**同一句话**内    | 只写「见下图」、无图号       |
| 书签指向页内具体段落（Y坐标）    | 书签仅绑页码            |
| 纯文字参数表（可选中、可复制）    | 表格内嵌图片、表格做成截图/扫描图 |


---

## 3. 图注与图号（最重要）

系统靠**图注**和**正文图号引用**把插图挂到对应步骤。无图号时只能靠章节检索，难以在问答中按步骤展示。

### 图注（紧贴插图/表格下方，单独一行）

```
图 4-7 修改起弧时间图示
表 2-1 额定参数表
Figure 4-7 Arc start time illustration
Table 2-1 Rated parameters
```

- 行首 `图` / `表`，或 `Figure` / `Fig.` / `Table` + `章号-序号` + 说明（连字符支持 `-` / `–` / `—`）
- 不要用冒号代替空格（如 `图:4-7`、`Figure: 4-7`）
- 图注不进入正文检索，正文须单独引用图号

### 正文引用（写在步骤**同一句话**内，可跨页）

```
1. 将起弧时间设为 0.5 s（参考图 4-7）。
2. 检查扫描位置设置（参考图 4-8）。
```

常用写法：`（参考图 4-7）`、`如图 4-6 所示`、`参数见表 2-1`；英文：`see Figure 4-7`、`as shown in Fig. 4-6`、`refer to Table 2-1`。正文图号须与图注一致。

### 跨页与位置

1. **正文图号引用**（优先，可跨页）
2. **图注图号**反向匹配正文
3. **阅读顺序**兜底：同页挂正上方最近正文；跨页最多前 2 页

说明与图相距超过 2 页时，**必须在正文写图号**，不要依赖位置。

## 4. PDF 书签

正式手册须设**大纲/书签**，标题与正文一致：

- 1 级章 → 2 级节 → 3 级小节（如 `第一章` → `1.1` → `1.1.1`）
- 书签目标须为**页内具体位置**（页码 + Y 坐标），不要只填「第 N 页」
- 同页多小节时，各书签落在各自标题行，不要都堆在页顶
- 在 Acrobat / Foxit 中：滚动到目标行 →「设置目标」→「缩放到」选中段落
- 「第一章」之前的前置页可能被跳过；带点线目录样式的页面会自动忽略

---

## 5. 扫描件

优先提供可选中文本层 PDF。纯扫描件须清晰可辨，页内精细切分受限，至少保证书签页级章节正确。修改手册后须**重新上传**（同名自动重索引）才会更新问答内容。

---

## 6. 上传前检查

- [ ] PDF 书签完整，目标为页内具体位置（含 Y 坐标）
- [ ] 关键信息在正文主体，非仅页眉页脚或图上
- [ ] 每图单独一行图注：`图 X-Y 说明` / `表 X-Y 说明`
- [ ] 步骤句内写图号引用，一步骤一图号，与图注一致
- [ ] 参数表为文字表（非截图/扫描图），无表格内嵌图；正文可选中文本层
- [ ] 已重新上传或触发重新索引

---

## 7. 常见问题


| 问题                | 处理                                          |
| ----------------- | ------------------------------------------- |
| 答成错误章节 / 同页小节混在一起 | 补全书签；每个书签设到对应标题行（含 Y 坐标）                    |
| 参数检索不到            | 用可选中文本层的矢量文字表，勿用图片格式表格；关键数值在正文重复            |
| 插图很少出现或位置不对       | 补图注 `图 X-Y`；步骤**同一句**内写 `（参考图 X-Y）`；图号与图注一致 |
| 图在下一页能对上吗         | 可以，正文须在同一句写图号；超 2 页无图号则难关联                  |
| 封面搜不到             | 「第一章」前内容可能被跳过，调整书签结构                        |
| 修改后仍是旧内容          | 重新上传同名文件或触发重新索引                             |


插图缺少图注时，管理员可开启 `INGEST_CAPTION_ENABLED` 生成补充描述，但**不能替代**正文图号引用。
